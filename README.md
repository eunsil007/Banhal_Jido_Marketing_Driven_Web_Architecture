## 🎯 Service Strategy (기획 전략)
본 프로젝트는 단순한 정보 나열이 아닌, **'지역 관광 활성화'**와 **'합리적 소비'**라는 두 가지 비즈니스 목표를 마케팅 관점에서 풀어낸 프로토타입입니다.

* **Problem (문제 제기)**: 고물가로 인한 국내 여행 심리 위축 및 특정 지역 관광 편중 현상.
* **Solution (해결 방안)**: '반값(Economic Value)'과 '반하다(Emotional Value)'라는 중의적 메시지를 결합한 브랜드 경험 설계.
* **Target (대상)**: 10대부터 80대까지 전 세대가 거부감 없이 이용할 수 있는 직관적인 UI/UX 제공.

## 🖋️ Brand Experience & Copywriting (브랜딩)
기획 단계에서 수립한 브랜드 아이덴티티를 디자인 전반에 투영했습니다.

- **브랜드 네이밍**: '반할지도' - 중의적 표현을 활용해 브랜드의 기능과 감성을 동시에 전달.
- **슬로건 설계**: "당신의 마음에 반하고, 반값 가격에 반하다."
- **카피라이팅 전략**: 전 연령층이 이해하기 쉬운 일상 용어를 사용하여 서비스 진입 장벽을 낮춤.
- **시각적 언어**: 브랜드의 신뢰도를 높이는 모노크롬 에스테틱과 가독성 높은 레이아웃 채택.

## 🛠️ Tech Stack & Implementation (기술 구조)
기획자의 의도가 사용자에게 정확히 전달될 수 있도록 퍼포먼스 중심의 환경을 구축했습니다.

- **Design System**: Figma (컴포넌트 기반의 체계적인 UI 라이브러리 구축)
- **Typography**: Pretendard (모든 기기에서 균일한 브랜드 경험을 위한 최적화 폰트)
- **Framework**: Bootstrap 5 (그리드 시스템을 활용한 반응형 웹 구조 설계)
- **Publishing**: 웹 표준을 준수하는 HTML5/CSS3 퍼블리싱
